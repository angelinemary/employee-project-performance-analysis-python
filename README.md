# 👩‍💼 Employee & Project Performance Analysis Using Python (Pandas & NumPy)

## 📌 Objective
To analyze employee performance and project outcomes by cleaning, transforming, and combining multiple datasets.

## 📂 Dataset Description
- Employee Dataset → Employee details (ID, name, gender, city, age)
- Project Dataset → Project cost and status
- Seniority Dataset → Employee designation levels

## 🛠 Tools Used
- Python
- Pandas
- NumPy

## 📊 Key Tasks Performed
- Cleaned missing project cost values using running average
- Split and transformed employee data
- Merged multiple datasets into one
- Applied business rules for bonus, promotion, and demotion
- Calculated total project cost per employee

## 📈 Key Insights
- Project completion impacts employee incentives
- Failed projects affect designation levels
- Data cleaning improves analysis accuracy

## 🚀 Outcome
Demonstrates strong data cleaning, transformation, and analytical skills using Python.
